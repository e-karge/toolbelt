oathtool
========

Docker image for `oathtool` from the
[OATH Toolkit](https://www.nongnu.org/oath-toolkit/). To use run

```bash
docker run --rm -ti toolbelt/oathtool
```

See [here](http://www.nongnu.org/oath-toolkit/oathtool.1.html) for invocation
details.
